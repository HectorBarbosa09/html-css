Vamos começar a entender formas de destaques e de ênfase

Para as formas de destaques;

podemos destacar o termo usando a tag <b> , porém não é uma tag semântica.


Podemos destacar o termo usando a tag <strong>, sendo essa considerada uma tag semâmtica.


Para as formas de ênfase, existem 3 formas, sendo uma não semântica e duas semânticas quase iguais.

A forma não semântica damos ênfase ao termo usando a tag <i>

Agora, usando as formas semânticas temos como dar ênfase usando a tag <em>

E a terceira forma para dar ênfase, usamos o atalho ctrl+shift+P

Texto marcado

Podemos criar também um texto marcado usando a tag <MARK>


Aqui também teremos outro parte marcada.


Texto grande e pequeno

Aqui estamos criando um texto grande e também um texto pequeno


Texto deletado

Podemos marcar um texto excluído, para que ele seja lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois

Texto subrescrito

Para texto subrescritos, usamos o exemplo da matemática, x2+5, para colocar o x ao quadrado, usamos a tag <sup>

Texto subescrito

Para textos subescritos, usamos a química e a fórmula da água como exemplo, h2o, para colocar o 2 embaixo, usamos a tag <sub>
